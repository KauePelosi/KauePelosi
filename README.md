### Projects

**[mini-alloc](https://github.com/KauePelosi/mini-alloc)** &nbsp;`C++`  
Custom memory allocator implemented from scratch. Includes benchmarks comparing stack allocation, `malloc/free`, and the linear allocator. Built to understand memory management at the implementation level — not just the theory.

**[mini-shell](https://github.com/KauePelosi/mini-shell)** &nbsp;`C++`  
A minimal Unix shell. Parses and executes commands, supports basic piping. The kind of project that makes you appreciate how much work `bash` is doing.

**[leetcode-cpp](https://github.com/KauePelosi/leetcode-cpp)** &nbsp;`C++`  
Structured solutions to algorithmic problems, organized by topic and difficulty. Comments explain the reasoning, not just the code.
